# Keyboard Shortcuts

Shortcuts are active when focus is not inside a text input, textarea, or dropdown.

## Navigation

| Key | Action |
|---|---|
| `1` | Go to Today |
| `2` | Go to Inbox |
| `3` | Go to Calendar |
| `4` | Go to Planner |
| `Cmd/Ctrl+F` | Go to Search |

## Tasks

| Key | Action |
|---|---|
| `N` | Open New Task modal |
| `Escape` | Deselect the current task / close the Details panel |

## In the Task Modal

| Key | Action |
|---|---|
| `Cmd/Ctrl+Enter` | Save the task |
| `Escape` | Close the modal without saving |

## In the Details Panel

| Key | Action |
|---|---|
| `Escape` | Cancel inline edit / close the panel |
| `Enter` | Confirm inline edit |
